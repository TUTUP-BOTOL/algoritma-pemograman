STRUKTUR DASAR ALGORITMA

Struktur dasar algoritma merupakan komponen yang digunakan untuk menyusun langkah-langkah penyelesaian masalah secara sistematis. Beberapa bagian penting dalam notasi algoritmik meliputi tipe dasar, ekspresi, harga (nilai), assignment, dan input/output.

PENJELASAN

1. Tipe Dasar

Tipe dasar adalah jenis data yang digunakan untuk menyimpan nilai. Contohnya Integer (bilangan bulat), Real/Float (bilangan pecahan), Character (karakter), dan Boolean (Benar atau Salah).

2. Ekspresi

Ekspresi adalah kombinasi variabel, konstanta, dan operator yang menghasilkan suatu nilai. Contoh: total = harga * jumlah

3. Harga (Nilai)

Harga atau nilai adalah isi yang dimiliki oleh sebuah variabel. Misalnya: umur = 19, maka nilai variabel umur adalah 19.

4. Assignment

Assignment adalah proses pemberian nilai kepada variabel menggunakan tanda panah atau simbol "=". Contoh: nama = "Ilham"

5. Input / Output

Input adalah data yang dimasukkan oleh pengguna, sedangkan output adalah hasil yang ditampilkan oleh program.

NOTASI ALGORITMIK (SKEMA)

Deklarasi

nama : string
umur : integer
nilai : real
status : boolean

Deskripsi

Input nama
Input umur
Input nilai

status ← nilai >= 75

Tampilkan nama
Tampilkan umur
Tampilkan nilai
Tampilkan status

CONTOH PROGRAM PYTHON 


# Input
nama = input("Masukkan nama : ")
umur = int(input("Masukkan umur : "))
nilai = float(input("Masukkan nilai : "))

# Ekspresi
status = nilai >= 75

# Output
print("\n===== DATA SISWA =====")
print("Nama   :", nama)
print("Umur   :", umur)
print("Nilai  :", nilai)
print("Lulus  :", status)

ALGORITMA, STRUKTUR, DAN KARAKTERISTIK

Algoritma merupakan urutan langkah-langkah logis dan sistematis yang digunakan untuk menyelesaikan suatu masalah sehingga menghasilkan keluaran yang diinginkan.

PENJELASAN MATERI

1. Algoritma

Algoritma adalah serangkaian langkah-langkah yang disusun secara logis dan sistematis untuk menyelesaikan suatu masalah.

Contoh : Membuat teh.

  • Siapkan gelas.
  • Masukkan teh dan gula.
  • Tuangkan air panas.
  • Aduk hingga tercampur.
  • Teh siap diminum.

2. Struktur Algoritma

Struktur algoritma merupakan pola dasar dalam penyusunan langkah-langkah program.

  • Sekuensial (urutan).
  • Percabangan (seleksi).
  • Perulangan (iterasi).

3. Karakteristik Algoritma

  • Memiliki input (masukan).
  • Menghasilkan output (keluaran).
  • Langkah-langkah jelas dan tidak ambigu.
  • Memiliki akhir (finite).
  • Efektif dan dapat dikerjakan.

CONTOH STRUKTUR ALGORITMA

1. Sekuensial

Input panjang
Input lebar
luas ← panjang × lebar
Tampilkan luas

--------------------------------

2. Percabangan

Input nilai
Jika nilai ≥ 75 maka
  Tampilkan "Lulus"
Jika tidak
  Tampilkan "Tidak Lulus"

--------------------------------

3. Perulangan

Untuk i = 1 sampai 5
  Tampilkan i

KARAKTERISTIK ALGORITMA

  1. Input : Memiliki data masukan yang akan diproses.
  2. Output : Menghasilkan informasi atau hasil.
  3. Definiteness : Setiap langkah harus jelas.
  4. Finiteness : Memiliki akhir atau berhenti setelah sejumlah langkah.
  5. Effectiveness : Langkah-langkah dapat dilakukan dan efisien.

CONTOH PROGRAM PYTHON 


# Input
nilai = int(input("Masukkan nilai : "))

# Percabangan
if nilai >= 75:
    status = "Lulus"
else:
    status = "Tidak Lulus"

# Output
print("Status :", status)

# Perulangan
print("\nBilangan 1 - 5")
for i in range(1, 6):
    print(i)
