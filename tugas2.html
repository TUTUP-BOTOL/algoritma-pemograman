ARRAY SATU DIMENSI

Array merupakan struktur data yang digunakan untuk menyimpan sekumpulan data yang memiliki tipe yang sama di dalam satu variabel dengan indeks tertentu.

1. Apa yang dimaksud dengan Array?

Array adalah kumpulan elemen atau data yang memiliki tipe data sama dan disimpan dalam satu variabel dengan indeks sebagai penunjuk posisi setiap elemen.

2. Jelaskan yang dimaksud dengan Array 1 Dimensi

Array satu dimensi adalah array yang hanya memiliki satu baris atau satu indeks. Setiap elemen diakses menggunakan satu indeks yang dimulai dari 0.

Contoh:

nilai = [80, 85, 90, 95]

3. Bagaimana cara mendeklarasikan array?

Deklarasi array dilakukan dengan menentukan tipe data kemudian diikuti nama array dan jumlah elemennya.

int[] nilai = new int[5];

4. Tipe data apa yang digunakan untuk mendeklarasikan array?

Array dapat menggunakan berbagai tipe data, antara lain:

  • int → bilangan bulat.
  • float atau double → bilangan pecahan.
  • char → karakter.
  • boolean → nilai benar atau salah.
  • String → teks atau kumpulan karakter.

5. Sebutkan operasi yang diketahui pada array!

  • Traversal (mengakses seluruh elemen array).
  • Pencarian (Searching).
  • Penyisipan data (Insertion).
  • Penghapusan data (Deletion).
  • Pengurutan data (Sorting).
  • Pembaruan data (Updating).

CONTOH PROGRAM ARRAY 1 DIMENSI (PYTHON) 


# Array satu dimensi
nilai = [80, 85, 90, 95, 100]

print("Daftar Nilai Siswa")

# Menampilkan seluruh elemen array
for i in range(len(nilai)):
    print("Indeks", i, "=", nilai[i])

# Mengakses elemen tertentu
print("\nNilai pertama =", nilai[0])
print("Nilai terakhir =", nilai[4])
