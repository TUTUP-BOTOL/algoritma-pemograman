JAWABAN TUGAS TIPE DATA DAN OPERATOR JAVA

1. Perbedaan Tipe Data Primitif dan Non-Primitif

Primitif Non-Primitif
Menyimpan nilai secara langsung. Menyimpan referensi objek.
Ukuran memori tetap. Ukuran memori dapat berubah.
Contoh: int, float, char, boolean. Contoh: String, Array, Object.

2. Kesalahan Penulisan Variabel

a. Int 5angka;

Salah karena nama variabel tidak boleh diawali angka dan penulisan tipe data harus huruf kecil.

Benar : int angka5;

b. String nama siswa;

Salah karena nama variabel tidak boleh mengandung spasi.

Benar : String namaSiswa;

c. Int static;

Salah karena "static" merupakan keyword (kata kunci) pada Java dan tipe data int harus ditulis huruf kecil.

Benar : int nilaiStatic;

4. Operator Aritmatika Java

Operator Fungsi
+ Penjumlahan
- Pengurangan
* Perkalian
/ Pembagian
% Sisa hasil bagi (modulus)

5. Contoh Operator Pembagian Bahasa Java 


public class Pembagian {

    public static void main(String[] args) {

        int angka1 = 20;
        int angka2 = 5;

        int hasil = angka1 / angka2;

        System.out.println("Hasil pembagian = "
                            + hasil);

    }

}
