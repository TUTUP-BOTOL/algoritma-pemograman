Sistem manajemen toko buku sederhana

Buat program Python untuk mengelola data buku di sebuah toko kecil —> mencakup input data, pencarian, pengurutan, dan laporan stok

Latar Belakang

Toko Buku "Aksara" memiliki koleksi 10 buku. Setiap buku memiliki judul, nama pengarang, harga (dalam rupiah), dan jumlah stok. Pemilik toko membutuhkan program Python untuk mengelola data tersebut agar lebih mudah mencari, mengurutkan, dan memantau kondisi stok.

Fitur Yang Harus Dibuat

1. Input data buku: Buat prosedur untuk mengisi data 10 buku ke dalam array. Validasi: harga harus > 0 dan stok tidak boleh negatif.
2. Tampilkan semua buku: Buat prosedur untuk mencetak semua data buku dalam format tabel rapi (nomor, judul, pengarang, harga, stok).
3. Hitung total nilai stok: Buat fungsi yang menghitung total nilai stok seluruh buku (harga x stok per buku, lalu dijumlahkan). Kembalikan nilainya.
4. Urutkan berdasarkan harga: Urutkan data buku berdasarkan harga dari termurah ke termahal menggunakan Bubble Sort atau Selection Sort.
5. Cari buku terurut by harga: Setelah data diurutkan berdasarkan harga, gunakan binary search untuk mencari buku dengan harga tertentu yang diinput pengguna.
6. Urutkan berdasarkan stok: Urutkan data buku berdasarkan jumlah stok dari terbanyak ke tersedikit menggunakan Insertion Sort.
7. Laporan stok menipis: Buat fungsi yang mengembalikan daftar buku dengan stok ≤ 3 (stok menipis). Tampilkan daftar tersebut beserta jumlahnya.

KODE PROGRAM 


    # =====================================
    # SISTEM MANAJEMEN TOKO BUKU SEDERHANA
    # =====================================
    
    buku_list = []
    
    
    # 1. INPUT DATA BUKU
    def input_buku():
        print("\n=== INPUT DATA 10 BUKU ===")
    
        for i in range(10):
            print(f"\nData Buku ke-{i+1}")
    
            judul = input("Judul Buku      : ")
            pengarang = input("Nama Pengarang  : ")
    
            # Validasi harga
            while True:
                harga = int(input("Harga (Rp)      : "))
                if harga > 0:
                    break
                print("Harga harus lebih dari 0!")
    
            # Validasi stok
            while True:
                stok = int(input("Jumlah Stok     : "))
                if stok >= 0:
                    break
                print("Stok tidak boleh negatif!")
    
            buku = {
                "judul": judul,
                "pengarang": pengarang,
                "harga": harga,
                "stok": stok
            }
    
            buku_list.append(buku)
    
    
    # TAMPILKAN SEMUA BUKU
    def tampilkan_buku(data):
        print("\n=== DATA BUKU ===")
    
        print("-" * 90)
        print(f"{'No':<5}{'Judul':<25}{'Pengarang':<20}{'Harga':<15}{'Stok':<10}")
        print("-" * 90)
    
        for i, buku in enumerate(data, start=1):
            print(
                f"{i:<5}"
                f"{buku['judul']:<25}"
                f"{buku['pengarang']:<20}"
                f"Rp {buku['harga']:<12}"
                f"{buku['stok']:<10}"
            )
    
        print("-" * 90)
    
    
    # HITUNG TOTAL NILAI STOK
    def hitung_total_nilai_stok(data):
        total = 0
    
        for buku in data:
            total += buku["harga"] * buku["stok"]
    
        return total
    
    # 
    def bubble_sort_harga(data):
        n = len(data)
    
        for i in range(n - 1):
            for j in range(n - i - 1):
                if data[j]["harga"] > data[j + 1]["harga"]:
                    data[j], data[j + 1] = data[j + 1], data[j]
    
    
    # BINARY SEARCH BERDASARKAN HARGA
    def binary_search_harga(data, target):
        kiri = 0
        kanan = len(data) - 1
    
        while kiri <= kanan:
            tengah = (kiri + kanan) // 2
    
            if data[tengah]["harga"] == target:
                return tengah
            elif data[tengah]["harga"] < target:
                kiri = tengah + 1
            else:
                kanan = tengah - 1
    
        return -1
    
    
    # URUTKAN BERDASARKAN STOK
    def insertion_sort_stok(data):
        for i in range(1, len(data)):
            key = data[i]
            j = i - 1
    
            while j >= 0 and data[j]["stok"] < key["stok"]:
                data[j + 1] = data[j]
                j -= 1
    
            data[j + 1] = key
    
    
    # LAPORAN
    def stok_menipis(data):
        hasil = []
    
        for buku in data:
            if buku["stok"] <= 3:
                hasil.append(buku)
    
        return hasil

        print("2. Total Nilai Stok")

        print("4. Cari Buku Berdasarkan Harga")
        print("5. Urutkan Berdasarkan Stok")
        print("6. Laporan Stok Menipis")
        print("7. Keluar")
    
        pilihan = input("Pilih menu : ")
    
        if pilihan == "1":
            tampilkan_buku(buku_list)
    
        elif pilihan == "2":
            total = hitung_total_nilai_stok(buku_list)
            print(f"\nTotal Nilai Stok = Rp {total:,}")
    
        elif pilihan == "3":
            bubble_sort_harga(buku_list)
            print("\nData berhasil diurutkan berdasarkan harga.")
            tampilkan_buku(buku_list)
    
        elif pilihan == "4":
            bubble_sort_harga(buku_list)
    
            harga_cari = int(input("Masukkan harga yang dicari: "))
    
            posisi = binary_search_harga(buku_list, harga_cari)
    
            if posisi != -1:
                buku = buku_list[posisi]
    
                print("\nBuku ditemukan:")
                print("Judul      :", buku["judul"])
                print("Pengarang  :", buku["pengarang"])
                print("Harga      :", buku["harga"])
                print("Stok       :", buku["stok"])
            else:
                print("Buku dengan harga tersebut tidak ditemukan.")
    
        elif pilihan == "5":
            insertion_sort_stok(buku_list)
            print("\nData berhasil diurutkan berdasarkan stok.")
            tampilkan_buku(buku_list)
    
        elif pilihan == "6":
            daftar = stok_menipis(buku_list)
    
            print("\n=== LAPORAN STOK MENIPIS ===")
    
            if len(daftar) == 0:
                print("Tidak ada stok menipis.")
            else:
                tampilkan_buku(daftar)
                print(f"Jumlah buku stok menipis: {len(daftar)}")
    
        elif pilihan == "7":
            print("Program selesai.")
            break
    
        else:
            print("Pilihan tidak valid!")

1. Input Data Buku

  • for i in range(10) digunakan karena jumlah buku yang harus dimasukkan adalah 10 buku, sehingga perulangan dilakukan tepat 10 kali.
  • while True digunakan untuk validasi input, agar pengguna tidak dapat memasukkan harga ≤ 0 atau stok negatif.
  • Data buku disimpan dalam dictionary karena setiap buku memiliki beberapa atribut (judul, pengarang, harga, stok).
  • Semua dictionary disimpan dalam list sehingga mudah diakses dan diolah.

kode python - Input Data Buku - 


    buku_list = []

        print("\n=== INPUT DATA 10 BUKU ===")

                harga = int(input("Harga (Rp)      : "))

                "judul": judul,

3. Hitung Total Nilai Stok

  • Nilai stok suatu buku diperoleh dari: Nilai Stok=Harga x Stok
  • Karena semua buku harus dihitung, digunakan perulangan for.
  • Variabel total digunakan sebagai akumulator untuk menjumlahkan seluruh nilai stok.

kode python - Menghitung Total Nilai Stok - 


    
# HITUNG TOTAL NILAI STOK
def hitung_total_nilai_stok(data):
    total = 0

    for buku in data:
        total += buku["harga"] * buku["stok"]

    return total

4. Urutkan Berdasarkan Harga

  • Soal meminta menggunakan Bubble Sort atau Selection Sort. Saya memilih Bubble Sort karena mudah dipahami dan cocok untuk jumlah data yang sedikit (10 buku).
  • Bubble Sort bekerja dengan membandingkan dua data yang bersebelahan lalu menukarnya jika tidak sesuai urutan.

kode python - Mengurutkan Berdasarkan Harga - 


    
# mengurutkan berdasarkan harga
def bubble_sort_harga(data):
    n = len(data)

    for i in range(n - 1):
        for j in range(n - i - 1):
            if data[j]["harga"] > data[j + 1]["harga"]:
                data[j], data[j + 1] = data[j + 1], data[j]

5. Cari Buku Terurut by Harga

  • Binary Search lebih cepat dibanding pencarian biasa karena langsung memeriksa bagian tengah data.
  • Syarat Binary Search adalah data harus sudah terurut terlebih dahulu.

kode python - Mencari Buku Berdasarkan Harga - 


    # BINARY SEARCH BERDASARKAN HARGA
    def binary_search_harga(data, target):
        kiri = 0
        kanan = len(data) - 1

        while kiri <= kanan:
            tengah = (kiri + kanan) // 2

            if data[tengah]["harga"] == target:
                return tengah
            elif data[tengah]["harga"] < target:
                kiri = tengah + 1
            else:
                kanan = tengah - 1

        return -1

6. Urutkan Berdasarkan Stok

  • Insertion Sort cocok untuk data berukuran kecil.
  • Cara kerjanya seperti menyusun kartu remi di tangan, yaitu menyisipkan data pada posisi yang tepat.

kode python - Mencari Buku Berdasarkan Harga - 


    # URUTKAN BERDASARKAN STOK
    def insertion_sort_stok(data):
        for i in range(1, len(data)):
            key = data[i]
            j = i - 1

            while j >= 0 and data[j]["stok"] < key["stok"]:
                data[j + 1] = data[j]
                j -= 1

            data[j + 1] = key

7. Urutkan Berdasarkan Stok

  • stok menipis adalah stok yang jumlahnya kurang dari sama dengan 3
  • Program memeriksa seluruh buku menggunakan perulangan for
  • Jika memenuhi syarat, buku dimasukkan ke dalam list hasil

kode python - Mencari Buku Berdasarkan Harga - 


    # LAPORAN
    def stok_menipis(data):
        hasil = []

        for buku in data:
            if buku["stok"] <= 3:
                hasil.append(buku)

        return hasil

7. Urutkan Berdasarkan Stok

input_buku() untuk memasukkan data 10 buku. Setelah itu, program menggunakan perulangan while True agar menu terus ditampilkan sampai pengguna memilih keluar.
Perintah input_buku() digunakan untuk mengisi data awal buku. Perulangan while True digunakan agar program tetap berjalan dan pengguna dapat menggunakan menu berulang kali tanpa harus menjalankan ulang program. Variabel pilihan digunakan untuk menyimpan menu yang dipilih pengguna.
Selanjutnya digunakan percabangan if-elif-else untuk menentukan fitur yang akan dijalankan berdasarkan pilihan pengguna.

Jika pengguna memilih menu 1, program memanggil prosedur tampilkan_buku() untuk menampilkan seluruh data buku dalam bentuk tabel.

Jika pengguna memilih menu 2, program memanggil fungsi hitung_total_nilai_stok(). Nilai yang dikembalikan oleh fungsi disimpan ke variabel total, kemudian ditampilkan kepada pengguna.

Jika pengguna memilih menu 3, program menjalankan algoritma Bubble Sort untuk mengurutkan buku berdasarkan harga dari termurah ke termahal. Setelah proses pengurutan selesai, data ditampilkan kembali.

Pada menu 4, data terlebih dahulu diurutkan menggunakan Bubble Sort karena Binary Search hanya dapat digunakan pada data yang sudah terurut. Setelah pengguna memasukkan harga yang dicari, program menjalankan fungsi binary_search_harga(). Jika data ditemukan, informasi buku ditampilkan. Jika tidak ditemukan, program menampilkan pesan bahwa data tidak tersedia.

Jika pengguna memilih menu 6, program memanggil fungsi stok_menipis() untuk mencari buku yang memiliki stok kurang dari atau sama dengan 3. Jika tidak ada data yang memenuhi syarat, program menampilkan pesan bahwa tidak ada stok menipis. Jika ada, daftar buku tersebut beserta jumlahnya akan ditampilkan.

Jika pengguna memilih menu 7, program menampilkan pesan selesai kemudian menjalankan perintah break untuk menghentikan perulangan while True, sehingga program berakhir.

else digunakan untuk menangani input yang tidak sesuai dengan pilihan menu yang tersedia. Jika pengguna memasukkan angka selain 1 sampai 7, program akan menampilkan pesan kesalahan dan kembali ke menu utama
