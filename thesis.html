Sistem manajemen toko buku sederhana

Buat program Python untuk mengelola data buku di sebuah toko kecil —> mencakup input data, pencarian, pengurutan, dan laporan stok

Latar Belakang

Toko Buku "Aksara" memiliki koleksi 10 buku. Setiap buku memiliki judul, nama pengarang, harga (dalam rupiah), dan jumlah stok. Pemilik toko membutuhkan program Python untuk mengelola data tersebut agar lebih mudah mencari, mengurutkan, dan memantau kondisi stok.

Fitur Yang Harus Dibuat

1. Input data buku: Buat prosedur untuk mengisi data 10 buku ke dalam array. Validasi: harga harus > 0 dan stok tidak boleh negatif.
2. Tampilkan semua buku: Buat prosedur untuk mencetak semua data buku dalam format tabel rapi (nomor, judul, pengarang, harga, stok).
3. Hitung total nilai stok: Buat fungsi yang menghitung total nilai stok seluruh buku (harga x stok per buku, lalu dijumlahkan). Kembalikan nilainya.
4. Urutkan berdasarkan harga: Urutkan data buku berdasarkan harga dari termurah ke termahal menggunakan Bubble Sort atau Selection Sort.
5. Cari buku terurut by harga: Setelah data diurutkan berdasarkan harga, gunakan binary search untuk mencari buku dengan harga tertentu yang diinput pengguna.
6. Urutkan berdasarkan stok: Urutkan data buku berdasarkan jumlah stok dari terbanyak ke tersedikit menggunakan Insertion Sort.
7. Laporan stok menipis: Buat fungsi yang mengembalikan daftar buku dengan stok ≤ 3 (stok menipis). Tampilkan daftar tersebut beserta jumlahnya.

                    # =====================================

                    buku_list = []


                    # 1. INPUT DATA BUKU
                    def input_buku():
                        print("\n=== INPUT DATA 10 BUKU ===")

                        for i in range(10):
                            print(f"\nData Buku ke-{i+1}")

                            judul = input("Judul Buku      : ")
                            pengarang = input("Nama Pengarang  : ")

                            # Validasi harga
                            while True:
                                harga = int(input("Harga (Rp)      : "))
                                if harga > 0:
                                    break
                                print("Harga harus lebih dari 0!")

                            # Validasi stok
                            while True:
                                stok = int(input("Jumlah Stok     : "))
                                if stok >= 0:
                                    break
                                print("Stok tidak boleh negatif!")

                            buku = {
                                "judul": judul,
                                "pengarang": pengarang,
                                "harga": harga,
                                "stok": stok
                            }

                            buku_list.append(buku)
